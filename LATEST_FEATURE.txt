✨ FEATURES IN DETAIL
════════════════════════════════════════════════════════════════════

1️⃣  CHAT HISTORY LIST
    • Shows all your conversations
    • Each entry displays:
      - Title (first message, truncated)
      - Date (YYYY-MM-DD format)
      - Message count
      - Last message preview
    • Click any chat to load it
    • Hover to see delete button

2️⃣  NEW CHAT BUTTON
    • Blue [+] button in header
    • Starts fresh conversation
    • Clears current messages
    • Resets repository selection

3️⃣  DELETE FUNCTIONALITY
    • Hover over any chat entry
    • Trash icon appears
    • Click to delete
    • Auto-starts new chat if deleting active one

4️⃣  ACTIVE CHAT INDICATOR
    • Blue background (#EFF6FF)
    • Blue border (#1268FB)
    • Shows which chat you're viewing

5️⃣  STATISTICS PANEL
    • Total Chats: All saved conversations
    • Today: Chats created today
    • Large blue numbers
    • Bottom of sidebar

🚀 HOW TO USE
════════════════════════════════════════════════════════════════════

START NEW CHAT:
   1. Click [+] button in sidebar header
   2. Current chat clears
   3. Type your first message
   4. New chat auto-saves

LOAD PREVIOUS CHAT:
   1. Browse chat history list
   2. Click any chat entry
   3. Chat becomes active (blue highlight)
   4. (Note: Demo mode - messages not fully persisted)

DELETE A CHAT:
   1. Hover over chat entry
   2. Trash icon appears
   3. Click trash icon
   4. Chat removed
   5. New blank chat starts if needed

AUTO-SAVE:
   • First message creates new chat session
   • Title = first 40 chars of message
   • Message count updates automatically
   • Preview shows last AI response
