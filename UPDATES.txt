🚀 HOW TO USE
════════════════════════════════════════════════════════════════

1. Start the dev server:
   npm run dev

2. Open browser:
   http://localhost:3000

3. Default view:
   • Chat tab is selected (first tab)
   • Welcome screen shows with suggestions
   • Click a suggestion or type to start

4. Chat:
   • Type message in textarea
   • Press Enter to send
   • AI responds with message
   • Conversation continues naturally
