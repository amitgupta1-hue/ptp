💡 HOW TO USE REPOSITORY SELECTOR
════════════════════════════════════════════════════════════════════

EXAMPLE 1: Single Repository
   1. Open chat (default tab)
   2. Click [+] button
   3. Select: posp_api_node
   4. Chip appears: [🔵 posp_api_node ×]
   5. Type: "How does authentication work?"
   6. Press Enter
   7. AI knows context is posp_api_node

EXAMPLE 2: Multiple Repositories
   1. Click [+] button
   2. Select: posp, pbp_admin
   3. Chips appear: [🟢 posp ×] [🟠 pbp_admin ×]
   4. Type: "Compare user models"
   5. Press Enter
   6. AI compares across both repos

EXAMPLE 3: Remove Repository
   Option A: Click X on chip
   Option B: Click repo in popup again
   Result: Chip removed, context cleared
